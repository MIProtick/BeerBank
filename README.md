# React Redux REST API Boilerplate for BeerBank

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ git clone git clone https://gitlab.com/MIProtick/beerbank.git
$ cd BeerBank
$ npm install
$ npm start
```

For production environments...

```sh
$ npm install
$ npm run build
```
