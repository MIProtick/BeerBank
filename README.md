# The Beer Bank 
A simple React-Redux demo project.

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ git clone git clone https://gitlab.com/MIProtick/beerbank.git
$ cd BeerBank
$ npm install
$ npm start
```
